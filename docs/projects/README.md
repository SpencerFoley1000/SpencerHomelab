# Projects

This directory tracks homelab projects from idea to completion.

## Structure

- `planned/` - Ideas or upcoming work that has not started.
- `active/` - Work currently being implemented.
- `completed/` - Finished projects with lessons learned.

## Project Documentation Standard

Each project should include:

- Purpose
- Goals
- Scope
- Architecture impact
- Implementation notes
- Testing / validation
- Security considerations
- Lessons learned
- Follow-up tasks

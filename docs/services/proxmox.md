# Proxmox

## Purpose

Proxmox is the planned virtualization platform for running homelab virtual machines and infrastructure services.

## Status

- Status: Planned / pending detailed documentation
- Criticality: High once deployed

## Planned Documentation

This page should be expanded when Proxmox is installed or configured.

Document:

- Host hardware role

- Storage layout
- Network bridges
- VLAN handling
- VM naming conventions
- Backup strategy
- Update process
- Recovery procedure

## Security Considerations

- Do not expose management interfaces directly to the public internet.
- Document management access using sanitized placeholders.
- Separate experimental or attacker-style workloads from trusted infrastructure.
